# **App Name**: ChemLearn Interactive

## Core Features:

- Chapter Navigation: Easy navigation between the Periodicity and Electrochemistry chapters.
- Lecture Video Display: Display lecture videos in a clear, responsive video grid layout.
- Simulation Video Display: Display cartoon simulation videos alongside lectures to enhance understanding.
- Presentation Slides Display: Embed and display presentation slides (PDF or iframe) next to videos for reference.
- Interactive Quiz: Implement a multiple-choice quiz section with 5 questions to test knowledge for each chapter.
- Responsive Design: Ensure the layout is responsive and adapts to different screen sizes (tablets and mobile devices).
- Interactive UI elements: Implement hover effects on buttons and cards utilizing the color palette.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) to bring a modern and energetic feel.
- Secondary color: Deep Purple (#7952B3) to create depth and sophistication.
- Accent color: Soft Pink (#FF69B4) for highlights and interactive elements.
- Font: 'Inter', a sans-serif font to bring a clean and professional aesthetic for both body text and headlines.
- Glassmorphism effects (semi-transparent backgrounds) to give a modern, layered look to sections.
- Subtle hover animations on buttons and cards using the Electric Blue, Deep Purple, and Soft Pink color palette.
